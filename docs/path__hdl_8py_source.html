74 def path_expand(path):
75  """
76  This function should return a full path no matter what input path is a directory or file.
77  Only expand, never check whether it's exist.
78  Three cases are considered:
79  1, path is already a rooted path on Linux or a path start with C, D, E, F..Z on windows, which can be directed accessed.
80  2, Path contain ~, on linux it should be expanded to /home/user, on windows it should be expanded to User's Document
81  folder.
82  3, If path starts with [a-zA-Z0-9], this kind of path should be assumed to be located in program's root data path.
83  *, Other path types are considered undefined. Undefined path can be returned, but should with an error log,
84  a expanded path should be expanded to DATA_ROOT/undefined/path to make sure other
85  paths are not polluted.
86  #TODO this function should be called thousand times during processing a normal daily analysis. current implementation is still too heavy.
87  :param path: a user specified path string
88  :return: a expanded path string which can be used by other functions.
89  """
90  path = os.path.expanduser(path)
91  if os.path.exists(path):
92  return path
